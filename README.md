## 🛠️ Technologies Used
*   **Database:** PostgreSQL / MySQL
*   **Language:** SQL
*   **Tools:** VS Code, Beekeeper Studio, Draw.io

## 📂 Project Structure
*   `QUERY.sql` : Contains all the SQL commands (DDL for table creation, DML for inserting data, and DQL for fetching results).
*   `README.md` : Project documentation and submission links.

## 🚀 Key Features Implemented
*   Created `Users`, `Matches`, and `Bookings` tables with appropriate Primary and Foreign Keys.
*   Applied Constraints (`UNIQUE`, `CHECK`) to maintain data integrity.
*   Implemented advanced queries using `JOIN` (INNER, LEFT), `COALESCE`, `LIKE/ILIKE`, Subqueries, Aggregation (`AVG`), and Pagination (`LIMIT`, `OFFSET`).
